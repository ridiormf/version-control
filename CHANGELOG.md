# Changelog

Todas as mudanças notáveis neste projeto serão documentadas neste arquivo.

O formato é baseado em [Keep a Changelog](https://keepachangelog.com/pt-BR/1.0.0/),
e este projeto adere ao [Semantic Versioning](https://semver.org/lang/pt-BR/).

## [1.0.0] - 2025-12-24

### Added

- Sistema inteligente de análise de commits
- Sugestão automática de versionamento semântico (MAJOR, MINOR, PATCH)
- Atualização automática de package.json
- Atualização automática de CHANGELOG.md
- Interface CLI interativa e colorida
- Comandos Git automáticos (commit, tag, push)
- Suporte para uso programático via API
- Documentação completa em português
- Palavras-chave em português e inglês para análise
- Validação de entrada do usuário
- Suporte para execução em git hooks
